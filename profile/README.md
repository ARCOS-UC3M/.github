# Listado de software y proyectos del grupo

Esta organización tiene como objetivo recopilar y mostrar los proyectos realizados o en desarrollo dentro del grupo.

Cada usuario sigue siendo responsable de gestionar sus propios repositorios, pero esta web actúa como un índice común que facilita la navegación, ofreciendo enlaces directos a los repositorios originales.

Recomendación: cada proyecto debería tener una Wiki en la que se describa el proyecto, proceso de instalación, ejemplo de funcionamiento, y, si queréis, un listado de publicaciones y trabajos relacionados.
